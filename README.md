# 🚀 Portfólio Pessoal — Filipe Moura

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minhas habilidades, projetos práticos e experiência na área de tecnologia, desenvolvimento de software e análise de dados.

---

## 👨🏻‍💻 Sobre Mim

Sou estudante de **Análise e Desenvolvimento de Sistemas (ADS)**, com foco em desenvolvimento de software, aplicações web/mobile e inteligência de dados. Possuo experiência com tecnologias modernas do ecossistema JavaScript/TypeScript, além de práticas em modelagem de dados e SQL.

---

## 🛠️ Tecnologias Utilizadas no Projeto

- **Frontend:** React, TypeScript, Tailwind CSS
- **Ferramentas & Build:** Vite, Git, GitHub
- **Hospedagem:** Vercel

---

## 📌 Principais Destaques do Portfólio

- **Apresentação Profissional:** Resumo das competências e trajetória em ADS.
- **Projetos em Destaque:** Exibição interativa de projetos práticos e aplicações.
- **Download de Currículo:** Acesso direto para baixar o CV em formato PDF.
- **Contatos Directos:** Links para redes profissionais e canais de comunicação.
